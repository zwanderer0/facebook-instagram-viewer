Get Facebook User Access Token

Method 1: Graph API Explorer (Recommended)

  1. Open Graph API Explorer →
  2. In the top right, select your app: 1068046521647182
  3. Click on "User or Page" dropdown and select "Get User Access Token"
  4. Select these permissions:
  5. Click "Generate Access Token"
  6. Log in and approve the permissions
  7. Copy the generated token

Method 2: Direct OAuth Login

Click the button below to log in with Facebook and get your access token:

Login with Facebook

After login:

  1. You'll be redirected to a Facebook success page
  2. Look at the URL - it will contain: #access_token=YOUR_TOKEN_HERE
  3. Copy everything between access_token= and &

Method 3: Access Token Tool

Open Access Token Tool →

This tool shows all your current access tokens for different apps and pages.

Update Your .env.local File

Once you have your user access token, update your .env.local file:

# Facebook/Instagram Graph API Configuration
FACEBOOK_APP_ID=1068046521647182
FACEBOOK_APP_SECRET=4ef4431231c770c7da966bbd31d3e246
FACEBOOK_ACCESS_TOKEN=[PASTE YOUR NEW USER TOKEN HERE]

Note: User access tokens expire. For production use, implement token refresh or use long-lived tokens.
